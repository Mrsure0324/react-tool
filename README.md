# React Tool Demo

基于 `@umijs/max` 的 React 工具集合项目。

## 🚀 新功能：Figma to Code 工作流

将 Figma 设计稿自动转换为 React 组件代码！

### 快速体验

1. 启动项目：`pnpm dev`
2. 访问：http://localhost:8000/figma-to-code
3. 输入你的 Figma API Key 和设计稿链接
4. 一键生成 React 代码

### 功能特性

- ✅ 支持 Figma API 直接获取设计稿
- ✅ 自动解析设计结构和样式
- ✅ 生成 TypeScript + React 代码
- ✅ 保留 Flexbox 布局和样式
- ✅ 支持代码预览、复制、下载
- ✅ 本地存储 API Key

详细文档请查看：[FIGMA_TO_CODE.md](./FIGMA_TO_CODE.md)
